The writer's 33-heist slate, mapped onto the phased family plan. Answer to
"defer until after the phases?": **defer the build-out, not the mapping** —
the slate now drives which families each phase prioritizes, so nothing we
build is off-slate. Coverage today is already strong.

## Slate-driven design updates (recorded now)
1. **Mission-length tiers: 20 / 30 / 40 min** (small/medium/large) replace
   the Package's single 7-15 min window. Mission LENGTH stays downstream
   (objective mechanics own it), but SITE SCALE + route structure must
   support the tier — small = 1-3 buildings (current scale is right),
   medium = campus/row sites, large = mega-structure heroes.
2. Every family below carries its slate names so theming (Zoo) and specs
   pull toward the actual missions.
